## 번역 대상

- `resume` (스트리밍 일시 중단 이후 렌더링 재개 관련 문서)
- `resumeToPipeableStream` (Node.js Pipeable Stream 기반 재개 API 문서)

## 작업 내용

- React 한글 문서의 `resume` 및 `resumeToPipeableStream` 관련 문서를 번역했습니다.
- 기존 문서의 용어와 문체를 맞춰 번역 일관성을 유지했습니다.
- API 설명, 사용 맥락, 코드 예시 표현이 자연스럽게 전달되도록 문장을 다듬었습니다.

## 기여 포인트

- 서버 렌더링 문서 영역의 한글 접근성을 개선했습니다.
- 영문 원문의 의미를 유지하면서 한국어 독자 관점에서 이해하기 쉬운 표현으로 정리했습니다.
- 문서 품질(정확성/가독성/용어 통일)을 기준으로 번역을 보강했습니다.

## 결과

- PR을 통해 번역 내용이 리뷰 가능한 형태로 제출되었습니다.
- React 한국어 문서의 SSR 관련 참고 자료 확장에 기여했습니다.
