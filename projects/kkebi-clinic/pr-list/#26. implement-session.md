## #26 : 상담 세션 리스트, 상담 세션 및 상담 내용 요약(Summary) 페이지 구현

작업 링크 📎 : https://github.com/SungHyun627/KKEBI-clinic/commit/4fe61ca4

### 💡 다음 이슈를 해결했어요.

- 상담 세션 목록 페이지 기능 구현
  - 예정/완료 탭 구성
  - 리스트/뷰 상태 처리 및 캘린더 전환 버튼 상태 정리
  - 세션 시작 진입 플로우 연결
- 상담 일정 변경(Reschedule) 기능 구현
  - 일정 변경 모달 UI/상호작용 구현
  - 날짜 선택 + 시작/종료 시간 선택(TimeWheelPicker)
  - 변경사항 기반 버튼 활성화/비활성화
  - 일정 변경 API 연동 및 성공 토스트 처리
- 상담 세션 종료 후 요약 페이지(`/[locale]/session/[id]/summary`) 1차 구현 및 명세 기반 UI 구성
- 요약 데이터 조회 방식을 `sessionStorage` 기반에서 `sessionId` 기반 API 조회로 전환
- 요약 제출 플로우 구현
  - `react-hook-form` 기반으로 제출 상태/값 통합
  - 필수 조건(위험 수준, 다음 상담 시기, 권장 미션, 예약값 유효성)에 따른 CTA 활성화
  - 제출 API 연동 후 대시보드 이동 및 토스트 처리
- 요약 페이지 섹션 컴포넌트화
  - 완료 카드, AI 요약, 감정 패턴, 인지왜곡, 북마크된 순간, 상담사 평가, 권장 미션, 다음 상담 예약
- 다운로드 기능 구현/정리
  - TXT, PDF, 녹음 다운로드 로직 분리
- 다국어(i18n) 정비
  - summary 전용 번역 키 추가(ko/en)
  - summary 컴포넌트 하드코딩 문구 치환
- 리팩토링
  - `useSessionSummaryPage` 훅으로 비즈니스 로직 분리
  - summary 타입 공통화(`SummarySubmitFormValues`, API 응답 타입 등)
  - 미션 상수/날짜 포맷 유틸 분리
