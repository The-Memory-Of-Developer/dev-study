## #42 : summary 제출 오류 처리 및 실시간 요약 표시 로직 개선

작업 링크 📎 : https://github.com/SungHyun627/KKEBI-clinic/commit/f351abad

### 💡 다음 이슈를 해결했어요.

- summary 조회 시 상담 시간이 `elapsedSeconds` 기준으로 `분/초` 형태로 표시되도록 수정했어요.
- summary 제출 실패 시 영어 환경에서 한글 에러 문구가 노출되던 문제를 보완했어요.
- summary 제출 중 내부 서버 오류 메시지(`internal server error`/`서버 내부 오류`)가 발생하면, 토스트를 띄운 뒤 `/{locale}/sessions`로 이동하도록 처리했어요.
- 실시간 상담 화면의 Live summary 노출 조건을 개선했어요.
  - 대화가 1개 이상일 때만 요약이 보이도록 변경
  - 요약 문구를 “현재 감정 + 누적 대화 수” 형태로 정리
  - 스피커 영역에는 최근 2개 대화 요약이 표시되도록 정리
- 영어 환경에서 Live summary가 한글로 보일 수 있던 케이스를 보완했어요.
  <br><br>

### 💡 이슈를 처리하면서 추가된 코드가 있어요.

- summary duration 표기를 위한 i18n 키 추가
  - `messages/ko.json`: `summary.durationSecondUnit = "초"`
  - `messages/en.json`: `summary.durationSecondUnit = " sec"`
- 실시간 요약/감정 표시 로직 및 문구 계산 유틸 보강
  <br><br>

### 💡 필요한 후속작업이 있어요.
